# Certifications Repository

This repository gathers certificates completed as part of my professional learning and continuous development.

Most of the certificates were completed through the Next learning platform and were issued by providers such as Udemy and Pluralsight. The main focus areas are Artificial Intelligence, Machine Learning, Prompt Engineering, and Cybersecurity.

## Purpose

The goal of this repository is to provide a simple and organized way for professors and other reviewers to consult the certificates referenced in my CV.

## Contents

The certificates included here document completed training in areas such as:

- Artificial Intelligence
- Machine Learning
- Prompt Engineering
- Generative AI
- Cybersecurity
- Related technical topics

## Notes

- The certificates are shared only as proof of completion of the corresponding training.
- They do not contain confidential company information.
- File names may vary depending on the original issuing platform.

## CV Reference

This repository is intended to complement my CV by providing direct access to supporting certification documents.
